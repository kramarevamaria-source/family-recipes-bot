# Family Recipes Bot

Telegram-бот для автоматической публикации рецептов с картинкой.

## Переменные Railway

- `TELEGRAM_BOT_TOKEN` — токен от BotFather
- `TELEGRAM_CHANNEL` — адрес канала, например `@recipes_family`
- `TIMEZONE` — `Europe/Moscow`
- `POST_TIMES` — `09:00,14:00,19:00`
- `POST_ON_START` — `true` для тестовой публикации сразу после запуска

Бот должен быть добавлен в канал как администратор с правом публикации сообщений.
